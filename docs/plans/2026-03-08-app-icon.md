# App Icon Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Adicionar ícone "PR" com gradiente roxo/azul como favicon do browser e branding na sidebar da aplicação.

**Architecture:** Criar um SVG estático para o favicon e um componente React `PRIcon` reutilizável. Integrar o componente no `SidebarHeader` do `AppSidebar` acima do dropdown de usuário.

**Tech Stack:** SVG, React, Tailwind CSS, shadcn/ui Sidebar

---

### Task 1: Criar favicon SVG

**Files:**
- Create: `public/favicon.svg`
- Modify: `app/root.tsx`

**Step 1: Criar o arquivo `public/favicon.svg`**

```svg
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 32 32" width="32" height="32">
  <defs>
    <linearGradient id="grad" x1="0%" y1="0%" x2="100%" y2="0%">
      <stop offset="0%" stop-color="#7C3AED"/>
      <stop offset="100%" stop-color="#2563EB"/>
    </linearGradient>
  </defs>
  <rect width="32" height="32" rx="7" fill="#09090B"/>
  <text
    x="16"
    y="22"
    text-anchor="middle"
    font-family="system-ui, -apple-system, sans-serif"
    font-weight="700"
    font-size="16"
    fill="url(#grad)"
  >PR</text>
</svg>
```

**Step 2: Adicionar link do favicon SVG em `app/root.tsx`**

Dentro do `<head>`, após o `<Meta />` e antes do `<Links />`, adicionar:

```tsx
<link rel="icon" type="image/svg+xml" href="/favicon.svg" />
<link rel="icon" type="image/x-icon" href="/favicon.ico" />
```

Obs: o `.ico` existente fica como fallback para browsers antigos.

**Step 3: Verificar visualmente**

Rodar `npm run dev` e confirmar que o ícone aparece na aba do browser.

**Step 4: Commit**

```bash
git add public/favicon.svg app/root.tsx
git commit -m "feat(icon): add PR gradient SVG favicon"
```

### Task 2: Criar componente React `PRIcon`

**Files:**
- Create: `app/components/pr-icon.tsx`

**Step 1: Criar o componente**

```tsx
type Props = {
  size?: number
  className?: string
}

export function PRIcon({ size = 32, className }: Props) {
  return (
    <svg
      xmlns="http://www.w3.org/2000/svg"
      viewBox="0 0 32 32"
      width={size}
      height={size}
      className={className}
    >
      <defs>
        <linearGradient id="pr-icon-grad" x1="0%" y1="0%" x2="100%" y2="0%">
          <stop offset="0%" stopColor="#7C3AED" />
          <stop offset="100%" stopColor="#2563EB" />
        </linearGradient>
      </defs>
      <rect width="32" height="32" rx="7" fill="#09090B" />
      <text
        x="16"
        y="22"
        textAnchor="middle"
        fontFamily="system-ui, -apple-system, sans-serif"
        fontWeight="700"
        fontSize="16"
        fill="url(#pr-icon-grad)"
      >
        PR
      </text>
    </svg>
  )
}
```

**Step 2: Commit**

```bash
git add app/components/pr-icon.tsx
git commit -m "feat(icon): add PRIcon reusable React component"
```

### Task 3: Integrar PRIcon no header da sidebar

**Files:**
- Modify: `app/components/app-sidebar.tsx`

**Step 1: Importar o componente `PRIcon`**

No topo de `app/components/app-sidebar.tsx`, adicionar:

```tsx
import { PRIcon } from "~/components/pr-icon"
```

**Step 2: Adicionar bloco de branding no `SidebarHeader`**

Dentro de `<SidebarHeader>`, **antes** do `<DropdownMenu>` existente, adicionar:

```tsx
<div className="flex items-center gap-2 px-2 py-1">
  <PRIcon size={24} />
  <span className="text-sm font-semibold">PR Dashboard</span>
</div>
```

O resultado final do `<SidebarHeader>` deve ser:

```tsx
<SidebarHeader>
  <div className="flex items-center gap-2 px-2 py-1">
    <PRIcon size={24} />
    <span className="text-sm font-semibold">PR Dashboard</span>
  </div>
  <DropdownMenu>
    {/* ... dropdown existente sem alterações ... */}
  </DropdownMenu>
</SidebarHeader>
```

**Step 3: Verificar visualmente**

Rodar `npm run dev`, fazer login e confirmar que o logo aparece no topo da sidebar, acima do dropdown de usuário.

**Step 4: Commit**

```bash
git add app/components/app-sidebar.tsx
git commit -m "feat(sidebar): add PRIcon branding to sidebar header"
```
